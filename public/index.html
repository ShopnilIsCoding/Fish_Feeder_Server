API Endpoints

GET /api/health
Health check endpoint - returns server status
GET /api/devices/:deviceId/state
Get current state of a specific device
GET /api/devices/:deviceId/events
Get event history for a device (limit: 1-200)
POST /api/devices/:deviceId/schedule
Update feeding schedule (CSV format: HH:MM,HH:MM)
POST /api/devices/:deviceId/config
Update device configuration (servo angles, OLED, notifications)
POST /api/test-email
Send a test email notification
